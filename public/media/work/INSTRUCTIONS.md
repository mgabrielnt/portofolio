# Work Media

Use the same name as the slug in src/data/projects.ts.

Card image:
- Always uses public/media/work/images/[slug].webp

Popup media:
- Set popup to "image" or "video" in src/data/projects.ts
- Image popup uses public/media/work/images/[slug].webp
- Video popup uses public/media/work/videos/[slug].mp4
- Video poster still uses public/media/work/images/[slug].webp

PDF media:
- Put PDF files in public/media/work/pdf/[slug].pdf

Example for stockforecast:
- public/media/work/images/stockforecast.webp
- public/media/work/videos/stockforecast.mp4
- public/media/work/pdf/stockforecast.pdf
