# AbsensiYOLO — Cinematic Product Demo Video Prompt

## Master prompt

Create a premium cinematic product demo video for a web application called
**AbsensiYOLO**, an AI-powered attendance and ownership detection dashboard.
Use the provided screenshots as the main visual reference and preserve the
original interface identity: dark futuristic UI, deep navy and black background,
subtle red accent glow, rounded glassmorphism cards, thin borders, elegant data
tables, upload panels, and checklist panels with green status highlights.

The video should feel like a high-end AI SaaS launch film, mixing cinematic
screen capture, smooth cursor choreography, elegant UI transitions, subtle
parallax motion, ambient lighting, soft glow reflections, and dramatic product
reveal pacing. The tone must feel advanced, efficient, futuristic, reliable, and
visually premium.

Show the complete user journey from opening the product, understanding the
dashboard, uploading images, processing batch files, analyzing video, using live
webcam detection, reviewing checklist progress, then exporting the final
detection log. The result should feel like an intelligent computer vision
platform for real operational use.

Open with a dramatic macro shot of the brand name **AbsensiYOLO**. Show the
subtitle line: **Realtime YOLO attendance asset control** and the descriptive
line: **AI ownership detector for photo, batch upload, video, webcam, checklist,
and export log.** Let the hero section emerge from darkness with a slow
cinematic zoom and subtle red ambient glow.
